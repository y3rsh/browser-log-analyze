# browser-log-analyze

## Getting Started

```sh
brew install chromedriver
brew install pyenv
brew install pipenv
```

###  https://docs.pipenv.org/

### Install from the Pipfile

```sh
pipenv install
```

### Enter the created venv

```sh
pipenv shell
```

### Exit the venv

```sh
exit
```
